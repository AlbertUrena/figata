# Convencion de assets del menu

Estructura objetivo:

- `assets/menu/cards/`
- `assets/menu/modal/`
- `assets/menu/hover/`
- `assets/menu/gallery/`
- `assets/menu/placeholders/`

Notas:
- El lookup oficial siempre viene de `data/media.json`.
- `cards`: recomendado 600-900px de ancho.
- `modal`: recomendado 1200-2000px de ancho.
- `hover`: opcional, se usa fallback a `card` si falta.
- `gallery`: reservado para carousel futuro.
